## Структура проекта

- **task1/** — регистрационная форма (RegistrationForm) с обработкой событий и валидацией.
- **task2/** — управление статьями с компонентами ArticleManager, AddArticle, ArticleList, ArticleItem.

## Task 1: RegistrationForm

- Реализован контролируемый компонент формы.
- Поля: name, email, age.
- Валидация:
  - Имя: обязательно, минимум 2 символа.
  - Email: обязательно, корректный формат.
  - Age: обязательно, ≥18 лет.
- Ошибки отображаются под соответствующими полями.
- Предотвращена стандартная отправка формы (`preventDefault`).
- При успешной отправке отображается сообщение: `Registration successful!`.

## Task 2: ArticleManager & Reusable Components

- Компоненты:
  - **ArticleManager** — хранит состояние статей, обработчики добавления и удаления.
  - **AddArticle** — форма для добавления статьи.
  - **ArticleList** — список статей.
  - **ArticleItem** — отдельная статья с управлением видимостью содержания (`isOpened`).
- Использованы паттерны:
  - Разделение на переиспользуемые компоненты.
  - Локальное состояние для управления отображением каждой статьи.
  - Передача данных и функций через props.
