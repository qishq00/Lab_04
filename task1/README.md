# Task 1: RegistrationForm

## Цель

Создать форму регистрации с использованием обработки событий React и валидации данных.

## Реализованные функции

- Контролируемые поля ввода с использованием useState
- Обработчики событий onChange и onSubmit
- Функции валидации:
  - validateName()
  - validateEmail()
  - validateAge()
- Отображение ошибок под полями
- Использование preventDefault() для предотвращения перезагрузки страницы
- Отображение сообщения об успешной регистрации
- Очистка формы после успешной отправки

## Подход обработки событий

React synthetic events используются для обработки действий пользователя.

Каждое поле обновляет состояние через onChange.

Форма использует onSubmit и preventDefault() для контроля отправки.
